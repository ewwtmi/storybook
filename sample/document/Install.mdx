# 導入方法



## 事前準備

1. Node.jsをインストールしておく
2. プロジェクトフォルダをつくる（フォルダ名にアンダースコアは禁止）
3. プロジェクトフォルダ内に、中身ほぼデフォルトの `package.json` を入れておく

### `package.json` を入れておく理由
Storybookインストール時に「 `package.json` ないけど大丈夫？」みたいなエラーが出る  
name/versionの項目だけでもいいので入れておくと回避できる

### `package.json` の例

```json
{
  "name": "storybook-sample",
  "private": true,
  "version": "1.0.0",
  "type": "module",    // ドキュメント階層をカスタマイズする場合、ここを追加するようにエラーが表示される
  "devDependencies": {
    "@storybook/addon-docs": "^10.0.6",
    "@storybook/html-vite": "^10.0.6",
    "storybook": "^10.0.6"
  },
  "scripts": {
    "storybook": "storybook dev -p 6006",
    "build-storybook": "storybook build"
  }
}
```



## Storybookのインストール

1. Git Bushなどでプロジェクトフォルダへ移動
2. `npx storybook@latest init --type html` を実行

> Storybook 初期化時に、 —type html を指定することで、HTML を動かすための最小限のセットアップが行えます。
> 
> <cite>引用元: https://dev.henry.jp/entry/storybook-for-html</cite>
