# Introduction

Storybook for HTML のサンプルです



## 概要
コンポーネント一覧やスタイルガイド的に使うだけなら、for HTMLのみで十分  
デフォルトの時点で拡大縮小・グリッド表示・寸法表示などの機能が使える

※該当する設定ファイル以外は、慣れるまでは出力されたまま使うのを推奨  
（内容が複雑すぎて、どれか外すと動作しなくなる・・・）
